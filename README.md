# 🛡️ SSH Reverse Tunnel Pro  For Home Server 

این اسکریپت به شما کمک می‌کند تا یک تونل معکوس پایدار بین اوبونتو و ویندوز یا گوشی موبایل برقرار کنید.

### ✨ ویژگی‌ها
- ✅ نصب خودکار پیش‌نیازها
- ✅ حذف سرور ایران برای کاهش هزینه‌ها
- ✅ ساخت سرور خانگی
- ✅ اجرا آسان
- ✅ ساخت سرویس سیستمی برای پایداری در پس‌زمینه
- ✅اجرا به صورت مولتی لوکیشن برای سرور خانگی با ایران


### 1.🚀 نحوه نصب و اجرا روی سرور Ubuntu 
برای راه اندازی سریع تونل، دستور زیر را در ترمینال اوبونتو کپی و اجرا کنید:


```bash
bash <(curl -Ls https://raw.githubusercontent.com/saeederamy/SSH-Reverse-Tunnel/refs/heads/main/Installv2.sh)
```
